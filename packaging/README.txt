Yohaku
======

A lightweight Windows 11 tray app that reserves a configurable margin around every
monitor, so maximised and snapped windows get a gap from the screen edges while
staying genuinely maximised.

Running it
----------
Extract this archive to a folder you will keep (not a temporary location), then run
yohaku.exe. It lives in the system tray; right-click the icon for options.

Yohaku is unsigned, so Windows SmartScreen may warn ("Windows protected your PC"):
choose "More info", then "Run anyway".
